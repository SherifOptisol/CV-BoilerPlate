# Computer Vision Boilerplate

## Getting Started
1. Clone the repository
2. Install dependencies using PDM:
   ```bash
   pdm install
   ```

3. Run tests to ensure everything is set up:
   ```bash
   pdm run pytest
   ```

4. Run the main pipeline:
   ```bash
   pdm run python src/main.py
   ```

## Features
- Image Preprocessing
- Simple CNN Model
- Utility Functions
